# Ansible-Role-MongoDB

![](https://github.com/QiaoHao9/ansible-role-mongodb/actions/workflows/galaxy.yml/badge.svg)

## Install

```shell
$ ansible-galaxy install qiaohao9.mongodb
```
## Supported

- [x] Debian 9
- [x] CentOS 7/8
- [ ] CentOS 7/8 SELinux

## Example

```yaml
- hosts: localhost
  roles:
    - qiaohao9.mongodb
```
